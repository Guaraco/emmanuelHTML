Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<html> Define el inicio del documento HTML. <html></html>
La etiqueta <html> es la raíz de un documento HTML y contiene todos los demás elementos.
<head> Contiene metadatos/información sobre el documento. <head><title>Título</title></head>
La etiqueta <head> contiene elementos como <title>, <meta>, <link>, etc.
<title> Define el título del documento. <title>Mi Página</title>
El título aparece en la pestaña del navegador y en los resultados de búsqueda.
<body> Contiene el contenido visible de la página. <body>Hola Mundo</body>
Todo lo que se muestra en la página va dentro de <body>.
<h1> a <h6> Encabezados de diferentes niveles. <h1>Título Principal</h1>
<h1> es el más importante y <h6> el menos importante.
<p> Define un párrafo. <p>Este es un párrafo.</p>
Se utiliza para agrupar texto en párrafos.
<a> Define un hipervínculo. <a href="https://www.ejemplo.com">Enlace</a>
El atributo `href` especifica la URL del enlace.
<img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción">
El atributo `src` especifica la ruta de la imagen y `alt` proporciona un texto alternativo.
<ul> Define una lista desordenada. <ul><li>Item 1</li></ul>
Los elementos de la lista se definen con <li>.
<ol> Define una lista ordenada. <ol><li>Item 1</li></ol>
Los elementos de la lista se numeran automáticamente.
<li> Define un elemento de lista. <li>Elemento 1</li>
Se utiliza dentro de <ul> o <ol> para definir elementos de lista.
<div> Define una sección en el documento. <div>Contenido</div>
Se utiliza para agrupar elementos y aplicar estilos.
<span> Define una sección en línea. <span>Texto</span>
Se utiliza para aplicar estilos a una parte del texto.
<br> Inserta un salto de línea. Texto<br>Nueva línea
No tiene etiqueta de cierre y se usa para saltos de línea.
<hr> Inserta una línea horizontal. <hr>
Se utiliza para separar contenido con una línea.
<table> Define una tabla. <table><tr><td>Celda</td></tr></table>
Las tablas se estructuran con <tr> para filas y <td> para celdas.
<tr> Define una fila en una tabla. <tr><td>Celda</td></tr>
Contiene celdas (<td>) o encabezados (<th>).
<td> Define una celda en una tabla. <td>Celda</td>
Representa una celda de datos en una tabla.
<th> Define una celda de encabezado en una tabla. <th>Encabezado</th>
Se utiliza para encabezados de columnas o filas.
<form> Define un formulario. <form action="/submit" method="post"></form>
Se utiliza para recopilar datos del usuario.
